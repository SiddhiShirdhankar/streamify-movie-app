# Streamify - Movie Search App

## Overview
Streamify is a React-based movie search application that allows users to search movies using the OMDB API.

## Features
- Search movies by title
- Display movie posters, title, and year
- Loading indicator
- No results handling
- Default movie suggestions on load

## Tech Stack
- React (Vite)
- JavaScript (ES6+)
- CSS
- OMDB API

## Setup Instructions

1. Clone the repository
2. Install dependencies:
   npm install
3. Create a .env file:
   VITE_OMDB_API_KEY=your_api_key
4. Run the app:
   npm run dev

## Learnings
- React Hooks (useState, useEffect)
- API integration
- Component-based architecture
- Environment variables handling
